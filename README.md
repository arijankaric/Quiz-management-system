## General info
This project is a simple web application that enables management and playing of quizzes.
	
## Technologies
Project is created with:
* Java(JSP, MVC principles)
* Javascript
* jQuery
* Apache Derby & Tomcat
	
[comment]:## Setup
To run this project, install it locally using npm:

```
$ cd ../lorem
$ npm install
$ npm start
```
